\doxysubsection{Detailed Description}
A structure representing a bit array. 

It contains the number of bits present in the array and the array of bits. The bits are stored in an array of int\textquotesingle{}s. I used an array of int to have less computations to do when oring or anding two arrays. It is not possible to go higher than an int otherwise some functions will not work.
