\doxysubsection{Detailed Description}
Functions to compute metrics on a \mbox{\hyperlink{structStream}{Stream}}. 

The metrics are defined in the paper. I encourage you to read it to understand the metrics as explaining them here would be redundant. Most of the functions have been named the same way, but some have been renamed to be more coherent with the implementation, if I changed the name, I will give the name of the metric in the paper. ~\newline
 The metrics documentation have been split per section of the paper and ordered in the same order as in the paper. {\ttfamily \#include \char`\"{}interval.\+h\char`\"{}}\newline
