SGA_TimesIterator SGA_Stream_key_instants ( const SGA_Stream * stream)

Get the key instants of a Stream. A key instant is a time instant where the graph changes. For each time interval [t_start, t_end[ received from this iterator, all induced graphs {G(t_start), ..., G(t_end-1)} are the same.
