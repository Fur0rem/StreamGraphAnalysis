size_t SGA_Stream_temporal_cardinal_of_node_set (const SGA_Stream *stream)
 Compute the temporal cardinal of node set of a Stream. It is noted |W| in the paper. It is defined as the sum of the duration of the presence of all nodes. For example, if the stream contains 2 nodes A present at [0, 2[ and B present at [1, 4[, the temporal cardinal of the node set is (2 - 0) + (4 - 1) = 2 + 3 = 5.
