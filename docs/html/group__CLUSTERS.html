Clusters

Detailed Description

Clusters in a stream.

A cluster in a stream is defined as a subset of the temporal nodes of the stream. The links included in the cluster are the subset of the links of the stream that have both their nodes in the cluster.

See also
Section 6 of the paper for more information.

Here's an example on how to use SGA to do so : examples/clusters.c
