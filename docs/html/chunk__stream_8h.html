chunk_stream.h File Reference

Detailed Description

A Stream that represents a chunk of a StreamGraph. It contains a subset of the nodes and links of the StreamGraph. It also contains a subset of the time present in the StreamGraph.
It is useful to work on a subset of the StreamGraph. For example, to work on a specific time interval or on a specific set of nodes.
It is optimised to work on a big chunk of the StreamGraph. If you want to work on a small chunk of the StreamGraph, maybe ChunkStreamSmall will be more efficient.
