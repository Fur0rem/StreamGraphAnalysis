Detailed Description

Implementation of the algorithm to find maximal cliques in a Stream.
The algorithm to find the maximal cliques is a temporal version of the Bron-Kerbosch algorithm. Special thanks to Alexis Baudin for letting me use his implementation. I chose the non-parallel with pivot version. You can find the original code at https://gitlab.lip6.fr/baudin/maxcliques-linkstream.
