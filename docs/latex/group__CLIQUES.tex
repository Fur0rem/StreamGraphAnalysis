\doxysubsubsection{Detailed Description}
Compute the maximal cliques of a stream. 

A clique in a stream is a set of nodes over a time interval where each node is connected to all the other nodes in the set during that time. A maximal clique is a clique that is not included in a larger clique. \begin{DoxySeeAlso}{See also}
Section 7 of the paper for more information.
\end{DoxySeeAlso}
