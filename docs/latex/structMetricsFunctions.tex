\doxysubsection{Detailed Description}
Table of functions to compute metrics on a \mbox{\hyperlink{structStream}{Stream}}. 

Different types of \mbox{\hyperlink{structStream}{Stream}}\textquotesingle{}s can define a specialisation of some of these functions to compute metrics in a more efficient way. If set to NULL, the default implementation will be used.
