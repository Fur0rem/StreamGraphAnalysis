\doxysubsection{Detailed Description}
Create Stream\+Graph\textquotesingle{}s and manipulate generic \doxylink{structStream}{Stream}\textquotesingle{}s. A Stream\+Graph is the main data structure to represent a stream graph. It contains all the information about the nodes, links, and its presence in time. ~\newline
 A \doxylink{structStream}{Stream} is a wrapper/interface around a Stream\+Graph. All the functions to access the data and metrics take a \doxylink{structStream}{Stream} as input, and not a Stream\+Graph. The interface asks for the bare minimum to access the data and compute metrics on it. Various actual implementations of \doxylink{structStream}{Stream}\textquotesingle{}s can change the way a Stream\+Graph is viewed. ~\newline
 The 4 main types of \doxylink{structStream}{Stream} are\+: 


\begin{DoxyItemize}
\item Full\+Stream\+Graph\+: A Stream\+Graph as is.
\item Link\+Stream\+: A Stream\+Graph where all nodes are present at all times.
\item Chunk\+Stream and Chunk\+Stream\+Small\+: A subset of the Stream\+Graph.
\item Snapshot\+Stream\+: A Stream\+Graph at a specific time interval. ~\newline
 All of these \doxylink{structStream}{Stream}\textquotesingle{}s use an underlying Stream\+Graph, but transform the data in different ways. ~\newline
 The \doxylink{structStream}{Stream}\textquotesingle{}s do not actually modify the underlying Stream\+Graph, they only provide a different access to it. Each implementation tries to do lazy evaluation and only compute the data when needed, and transform it on the fly. Each implementation can also specialise some implementations of metrics to be more efficient. You can also manipulate multiple \doxylink{structStream}{Stream}\textquotesingle{}s of the same Stream\+Graph at the same time.
