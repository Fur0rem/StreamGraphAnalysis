\doxysubsection{Detailed Description}
A cache of information about the 4 main sets of a \doxylink{structStream}{Stream}\+: the temporal nodes W, the temporal links E, the life span T, and the distinct nodes V. 

Some metrics share the same computation, for example, the average node degree, compactness, and density all need the cardinal of W. Instead of recomputing the same value multiple times, we store it in a cache, as all the values are constant for a given \doxylink{structStream}{Stream}.
